========================================================================
src/ — исходный код проекта NOM-HRMS-FGA
========================================================================

Назначение
----------
Каталог содержит весь оригинальный исходный код программного пакета для
деконволюции масс-спектров высокого разрешения (ВЭЖХ-МСВР) природного
органического вещества (ПОВ) / гуминовых веществ. Здесь реализован полный
конвейер обработки: загрузка и очистка спектров, поиск изотопно-меченых
серий (дейтерометилирование –COOH и дейтероацилирование –OH), назначение
брутто-формул в режиме [M–H]⁻, подсчёт функциональных групп и генерация
возможных молекулярных структур.

Состав
------
  app.py            Графическое приложение (tkinter): единая точка входа
                    в интерактивный режим работы с конвейером.
  configs/          Централизованная конфигурация проекта: научные
                    константы, параметры конвейера и пути к данным в
                    JSON-файлах + загрузчик с единой точкой импорта
                    (`from src.configs import CHEM, PIPELINE, PATHS`).
                    Подробности — в configs/README.md.
  core/             Ядро научного метода — обработка спектров, разбор
                    формул, модель молекулы/атомов, сборка фрагментов в
                    структуры, мост к RDKit и оркестрация конвейера
                    (ключевой модуль — core/pipeline.py).
  ui/               Вспомогательные элементы графического интерфейса:
                    оформление (тема), встраивание графиков matplotlib.
  structures/       Отображение и экспорт молекулярных структур в GUI
                    (карточки, диалоги, утилиты RDKit/Pillow).
  testing/          Инфраструктура сквозного «smoke»-прогона: запуск
                    конвейера по всем тестовым наборам, экспорт артефактов
                    (графики, гистограммы, структуры) и сводных отчётов.
  simulations/      Генерация синтетических тестовых наборов и подготовка
                    молекул-кандидатов (в т.ч. из PubChem).

  result_table.csv, van_krevelen.png — примерные артефакты работы
  конвейера (результат и диаграмма Ван-Кревелена), приложены для
  наглядности.

Связь с общим конвейером
------------------------
src/ — это реализация всего метода деконволюции ПОВ. Пользователь
запускает конвейер либо через графический интерфейс (src/app.py), либо в
пакетном режиме (`python -m src.core.pipeline --test`). Модули core/
выполняют научную работу; ui/ и structures/ обеспечивают визуализацию;
testing/ и simulations/ используются для валидации и воспроизводимости.
