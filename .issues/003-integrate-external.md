## Описание

В директории `external/usrednenie_spectrov_i_hromatogramm/` лежит сторонний код (GPL-3.0) для усреднения спектров и хроматограмм. Сейчас он никак не интегрирован в основной пайплайн.

Необходимо:

1. Провести аудит текущего состояния external-кода: структура, зависимости, лицензии
2. Определить, какие функции из него нужны в основном пайплайне
3. Создать слой интеграции (wrapper/adapter) в `src/external_bridge.py` или `src/io/external.py`
4. Убедиться, что интеграция опциональна (external требует своих зависимостей: scipy, scikit-learn, comtypes)

## Критерии приемки

- [ ] Проведён аудит external-кода, зафиксированы доступные функции
- [ ] Создан интеграционный слой с чётким API
- [ ] Интеграция опциональна (fallback при отсутствии зависимостей)
- [ ] Добавлены тесты для интеграционного слоя
- [ ] Документация: как использовать external-функции в пайплайне
