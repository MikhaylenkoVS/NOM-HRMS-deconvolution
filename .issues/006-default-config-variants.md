## Описание

Сейчас GUI инициализируется с одним набором параметров из `pipeline.json -> run_pipeline_defaults`. Добавить возможность выбора из нескольких предустановленных конфигураций.

## Возможные варианты конфигов

- **"Synthetic (по умолчанию)"** — текущие настройки для синтетических тест-сетов
- **"Реальные спектры (NOM)"** — оптимизированные параметры для реальных образцов ГВ
- **"Быстрый тест"** — урезанные диапазоны для максимальной скорости

## Критерии приемки

- [ ] В GUI добавлен выпадающий список/радиокнопки для выбора пресета
- [ ] При смене пресета все поля параметров обновляются
- [ ] Пресеты хранятся в конфигах (pipeline.json или отдельном файле)
- [ ] Пользователь может изменить любой параметр после выбора пресета
- [ ] Добавить как минимум 2 дополнительных пресета к дефолтному
