## Описание

Сейчас GUI реализован через Tkinter (src/app.py), что требует десктопного окружения и установки tkinter. Если реализация веб-интерфейса на базе FastAPI/Flask + HTML-шаблонов или Streamlit займёт не больше одного рабочего дня — сделать это.

## Критерии приемки

- [ ] Проведена быстрая оценка: реально ли уложиться в 1 день?
  - Если да → реализовать
  - Если нет → законсервировать задачу до v0.5
- [ ] Если реализовано: веб-интерфейс покрывает базовую функциональность:
  - Загрузка трёх CSV-файлов (исходный, dmet, dacet)
  - Настройка параметров пайплайна
  - Запуск анализа
  - Просмотр результатов (таблица, Van Krevelen, спектры)
- [ ] Инструкция по запуску веб-версии в README

## Примечание

Streamlit — самый быстрый вариант (минимальный код). FastAPI + Jinja2 — если нужен больший контроль.
