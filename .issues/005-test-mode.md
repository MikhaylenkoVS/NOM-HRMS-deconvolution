## Описание

Добавить режим тестирования, который позволит запускать пайплайн на синтетических тест-сетах без ручного выбора файлов в GUI или ручного ввода путей в CLI.

## Возможные реализации

1. **В GUI**: кнопка "Запустить тест" → автоматически загружает синтетический набор (set_01..05) и прогоняет пайплайн с дефолтными параметрами. Показывает результаты в той же таблице.
2. **В CLI**: флаг `--test` для `src/app.py` или `src/core/pipeline.py`, который запускает smoke-тесты.

## Критерии приемки

- [ ] Тест-режим запускается одним действием (клик или флаг)
- [ ] Используются синтетические наборы из `data/test_sets/`
- [ ] Результаты выводятся в удобном виде (таблица + графики для GUI, stdout/CSV для CLI)
- [ ] Тест-режим не зависит от внешних файлов пользователя
- [ ] Smoke-тест (`pytest tests/integration/test_app_smoke.py`) согласован с тест-режимом
