## Описание

На данный момент проект — набор Python-скриптов, запускаемых через `python src/app.py` или `python -m src.core.pipeline`. Для конечного пользователя (особенно не-программиста) это неудобно: нужно клонировать репозиторий, ставить Python, устанавливать зависимости вручную.

Необходимо сделать проект легко скачиваемым и запускаемым одной командой.

## Критерии приемки

- [ ] Проект упакован как pip-установимый пакет (pyproject.toml с настройками)
- [ ] Можно установить одной командой: `pip install git+https://github.com/...`
- [ ] После установки доступна точка входа (entry point), например `nom-analyzer` или `python -m src`
- [ ] Обновлена документация по установке (README, CODE_AVAILABILITY.md)
- [ ] `pip install -r requirements.txt` работает из коробки (сейчас работает, но проверить)
