## Описание

### Словарь элементов и масс (`src/configs/chemistry.json`)
Сейчас включает: H, C, N, O, S, P. Необходимо:
- Добавить элементы, релевантные для ПОВ/ГВ (например, Na, K, Ca, Mg, Fe, Al — если встречаются в масс-спектрах)
- Проверить точность моноизотопных масс по актуальным источникам (IUPAC)
- Добавить распространённые изотопы для корректного моделирования M+1/M+2

### Словарь фрагментов (`src/core/fragments.py`)
Сейчас содержит:
- Ациклические фрагменты (methylene..propynyl) — корректно
- 5/6/8/10-членные углеродные циклы (возможно, избыточны для ГВ)
- Конденсированные системы (naphthalene, anthracene) — релевантно
- Гетероциклы с N и O — частично релевантно
- Функциональные группы (cooh..I) — нужно проверить полноту

Необходимо:
- Провести обзор существующей литературы по классам веществ гуминовых веществ (ГВ)
- Скорректировать словарь фрагментов:
  - Удалить или пометить фрагменты, не характерные для ГВ (например, 8-членные циклы)
  - Добавить фрагменты, типичные для ГВ (фенольные структуры, хиноны, углеводные фрагменты)
  - Добавить crosslinking-фрагменты (мостиковые структуры)
- Проверить, что обновлённый словарь совместим с существующим алгоритмом перебора комбинаций (`fragment_combinations.py`)

### Критерии приемки

- [ ] Проведён литературный обзор по классам веществ ГВ (результаты — в `docs/`)
- [ ] `chemistry.json` обновлён: добавлены новые элементы, проверены массы
- [ ] `fragments.py` обновлён: скорректирован список фрагментов под ГВ-химию
- [ ] Все существующие тесты проходят с обновлёнными словарями
- [ ] `fragment_combinations.py` работает с обновлённым списком
