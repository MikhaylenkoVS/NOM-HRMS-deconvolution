## Описание

В коде могут быть места, где используются синтетические/упрощённые предположения, не соответствующие реальным объектам анализа — природному органическому веществу (ПОВ/ГВ). 

Необходимо:

1. Провести аудит кода на предмет:
   - Упрощённых моделей молекул
   - Диапазонов элементов, не покрывающих реальные классы ГВ
   - Алгоритмов, завязанных на синтетические тест-сеты
   - Жёстко зашитых допущений, не подходящих для реальных спектров
   - Заглушек или эвристик, не проверенных на реальных данных
2. Исправить найденные проблемы
3. Показать результат научному руководителю (ДС)
4. Запросить у ДС реальные спектры для тестирования

## Критерии приемки

- [ ] Составлен список всех "нереальных" моментов с указанием файлов и строк
- [ ] Каждый пункт либо исправлен, либо документирован как осознанное допущение
- [ ] Результаты показаны ДС, получен фидбек
