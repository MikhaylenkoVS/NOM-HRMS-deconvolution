# NOM HRMS Deconvolution

Проект для дешифровки масс-спектров NOM (HPLC-HRMS).

## Структура проекта

- src/ — исходный код (ядро алгоритмов, IO, утилиты)
- data/ref_data — маленькие эталонные наборы данных
- data/test_sets — наборы для regression-тестов
- tests/ — pytest-тесты
- tools/ — служебные скрипты и утилиты
- docs/ — документация (описания источников данных, архитектуры)

## External code
- `external/usrednenie_spectrov_i_hromatogramm` — сторонний проект по усреднению спектров и хроматограмм.
  - Используется как источник алгоритмов / вспомогательных функций.
  - Исходный автор и лицензия указаны в `external/LICENSE.txt`.
