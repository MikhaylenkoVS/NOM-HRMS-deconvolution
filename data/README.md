========================================================================
data/ — данные проекта NOM-HRMS-deconvolution
========================================================================

Назначение
----------
Каталог содержит справочные и синтетические (эталонные) данные, на которых
валидируется и воспроизводится метод деконволюции масс-спектров ПОВ.
Внешние экспериментальные данные для запуска не требуются: прилагаемые
синтетические наборы позволяют воспроизвести работу всего конвейера.

Состав
------
  ref_data/
      ref_molecules_all_pubchem_filtered.csv — справочный список молекул,
          отобранных из PubChem как «ПОВ-подобные» (используется при
          генерации тестовых наборов и как источник структур-кандидатов);
      van_krevelen_nom_like.png — диаграмма Ван-Кревелена справочного
          набора (визуальная характеристика химического пространства).

  test_sets/
      set_01 … set_05 — пять синтетических тестовых наборов. Каждый набор
      содержит:
          original.csv          — исходный спектр [M–H]⁻;
          deutermethylated.csv  — спектр после дейтерометилирования (–COOH);
          deuteroacylated.csv   — спектр после дейтероацилирования (–OH);
          molecules.csv         — эталонный список молекул набора;
          annotations.csv       — эталонная разметка (формулы, число групп);
          config.json           — параметры генерации набора;
          *.bak                 — резервные копии исходных файлов.

Связь с общим конвейером
------------------------
Наборы из test_sets/ подаются на вход конвейеру (src/core/pipeline.py) и
используются автоматическими тестами (tests/). Сравнение результата
деконволюции с эталонной разметкой (annotations.csv, molecules.csv)
подтверждает корректность метода: подсчёта –COOH/–OH и назначения формул.
Данные из ref_data/ применяются модулями src/simulations/ при построении
синтетических спектров.
