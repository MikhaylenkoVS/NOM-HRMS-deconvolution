# Источники данных NOM-HRMS-FGA

Проект использует три категории данных: реальные (научные), справочные и синтетические.
Реальные масс-спектры хранятся вне репозитория; в репозитории — только референсные и
сгенерированные тестовые данные.

## 1. Реальные масс-спектры

Формат: Thermo RAW (.raw) или конвертированные CSV.

- Источник: HPLC-HRMS анализы гуминовых веществ (образцы предоставлены научным
  руководителем).
- Режим: отрицательная ионизация ([M–H]⁻), диапазон масс ~100–800 Да.
- Каждый образец — три спектра: исходный, после дейтерометилирования (–COOH → –COOCD₃),
  после дейтероацилирования (–OH → –OCOCD₃).
- Хранятся локально, **не включены в репозиторий** (объём — сотни МБ на образец).

## 2. Справочные данные (`data/ref_data/`)

### `ref_molecules_all_pubchem_filtered.csv`

Набор из ~300 молекул, отобранных из PubChem по критериям «NOM-подобия» (элементный
состав C/H/O/N, H/C 0.3–1.8, O/C 0.0–0.8, масса 100–800 Да).

Колонки: идентификатор CID/compound_id, формула, SMILES, InChI, заряд, число
COOH- и OH-групп, флаг NOM-подобия.

Используется для:
- калибровки диапазонов поиска формул (pipeline.json — formula_search/ranges);
- валидации NOM-приоритизации (штраф за N, расстояния до NOM-поля);
- эталонных визуализаций (Van Krevelen).

### `van_krevelen_nom_like.png`

Диаграмма Ван-Кревелена (H/C vs O/C) для референсных молекул — иллюстрация
химического пространства типичного природного органического вещества. Может
использоваться в публикациях.

## 3. Синтетические тестовые наборы (`data/test_sets/`)

Пять наборов (`set_01` … `set_05`), сгенерированных скриптом
`src/simulations/generate_test_sets.py`. Каждый набор самодостаточен и содержит
эталонную разметку (ground truth).

### Структура набора (set_01)

| Файл | Описание |
|------|----------|
| `molecules.csv` | Список молекул с истинными количествами COOH/OH (14 колонок) |
| `original.csv` | Исходный спектр (`mass,intensity`) с ppm-ошибкой и шумом |
| `deutermethylated.csv` | Спектр после CD₃-метилирования (сдвиг +17.0345 Да на COOH) |
| `deuteroacylated.csv` | Спектр после CD₃CO-ацилирования (сдвиг +45.0294 Да на OH) |
| `annotations.csv` | Полная аннотация всех пиков: теоретические массы, типы спектров, ppm-ошибки, флаг сигнал/шум (16 колонок) |

### Генерация

Скрипт `src/simulations/generate_test_sets.py` на основе конфигурации `pipeline.json`
и референсных молекул создаёт синтетические спектры:

1. Вычисление точных моноизотопных масс по формуле.
2. Применение дериватизационных сдвигов (DELTA_CD3, DELTA_CD3CO)
   со всеми возможными степенями замещения.
3. Внесение ppm-ошибки (нормальное распределение, μ = 0, σ задаётся).
4. Добавление шумовых пиков (управляется `pipeline.json → test_mode → denoise`).
5. Запись `annotations.csv` с полной прослеживаемостью «формула → масса → пик».

### Использование

Тестовые наборы — основной инструмент валидации конвейера:

- **Unit-тесты** (`tests/unit/`) — проверка отдельных функций (денойз, назначение
  формул, поиск серий, приоритизация) на эталонных данных.
- **Интеграционные тесты** (`tests/integration/`) — сквозной прогон
  `run_pipeline()` и сравнение результата с `annotations.csv`.
- **Smoke-тесты** — быстрая проверка работоспособности на всех наборах.

Воспроизводимость: `pytest tests/ -q` из чистого окружения проходит на всех
пяти наборах без внешних данных (тестовые CSV включены в репозиторий).
