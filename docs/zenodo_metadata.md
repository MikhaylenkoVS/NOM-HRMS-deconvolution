## Description (abstract)

NOM-HRMS-FGA is a research software package and graphical user interface
for functional-group analysis of natural organic matter (NOM) from
high-resolution mass spectrometry (HRMS) data acquired in liquid
chromatography (HPLC) experiments. The software provides a reproducible
pipeline for importing chromatographic–mass spectrometric data, performing
peak detection and integration, assigning peaks to predefined functional
groups, and exporting structured results for downstream analysis.
