Назначение
----------
Каталог содержит проектную документацию: описание архитектуры конвейера,
источников данных, методики генерации синтетических масс-спектров,
первоначальные обзоры и планы разработки. Материалы предназначены для понимания
устройства метода и воспроизведения результатов.

Состав
------
  architecture.md            Архитектура программного пакета (v0.4.1):
                             устройство конвейера деконволюции, взаимодействие
                             модулей (ядро, интерфейс, тестирование, симуляции).

  data_sources.md            Источники данных: реальные масс-спектры, справочные
                             молекулы PubChem, синтетические тестовые наборы.

  test_spectra_generation.md Методика генерации синтетических тестовых
                             спектров (модель масс, дейтерометки –COOH/–OH,
                             внесение шума и погрешности масс).

  Классы веществ в составе   Первоначальный обзор: аналитические свидетельства,
  гуминовых веществ ...      химические критерии и классификация соединений
                             в составе гуминовых веществ.

  Библиография ...           Полный библиографический список по масс-
                             спектрометрии ГВ, дериватизации и алгоритмам
                             обработки спектров (138 источников).

  plans/
    v0.4-milestone-plan.md   План закрытия milestone v0.4 (✅ завершён).
    v0.4-additional-tasks.md Дополнительные задачи от научного руководителя
                             к v0.4 (✅ завершены).
    v0.5-formula-dropdown.md План фичи «выпадающий список формул»
                             (✅ реализовано в v0.4.1).

Связь с общим конвейером
------------------------
Документация описывает научную и программную логику метода, реализованного
в src/, и объясняет, как формируются эталонные наборы в data/test_sets/,
на которых проверяется корректность конвейера.
